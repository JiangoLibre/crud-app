# README do CRUD

Este é um projeto de CRUD (Create, Read, Update, Delete) que permite a manipulação básica de dados em um sistema.

## Funcionalidades

O CRUD possui as seguintes funcionalidades principais:

- **Criar**: Permite adicionar novos registros ao sistema. Os usuários podem preencher os campos relevantes e salvar as informações.

- **Ler**: Possibilita visualizar os registros existentes no sistema. Os usuários podem consultar os dados armazenados para obter informações relevantes.

- **Atualizar**: Permite modificar os registros existentes. Os usuários podem editar os campos e salvar as alterações.

- **Excluir**: Permite remover registros do sistema. Os usuários podem selecionar um registro específico e excluí-lo permanentemente.

Estas funcionalidades básicas permitem a manipulação dos dados de forma simples e intuitiva. O CRUD fornece uma interface amigável para que os usuários possam interagir com o sistema e gerenciar os registros de maneira eficiente.
